# Post Checklist - Before Posting

## ✅ Content Quality
- [ ] Show what I built/learned (concrete examples)
- [ ] Share my thinking (not just conclusions)
- [ ] Provide code/repo links if applicable

## ✅ Questions (5+ entry points)
- [ ] Technical implementation question
- [ ] Architecture decision question
- [ ] Trade-offs question
- [ ] Experience question ("what worked for you?")
- [ ] Open invitation ("what are you seeing?")

## ✅ Reference Links
- [ ] Tag people doing related work
- [ ] Link to repos/projects I like
- [ ] Explain WHY I find it interesting
- [ ] Ask specific questions about their approach
- [ ] Link to relevant threads/documentation

## ✅ Engagement Design
- [ ] Multiple perspectives invited (not prescriptive)
- [ ] Create space for dialogue
- [ ] Specific enough to be useful
- [ ] Open enough to invite variety

## ❌ Avoid
- [ ] Just broadcasting without questions
- [ ] Generic "thoughts?" (too vague)
- [ ] No references to related work
- [ ] Prescriptive "here's the answer" tone
